Overview of Palo Alto Networks Products

Strata: Enterprise Security

  • Next-Generation Firewalls (NGFWs):
    • PA-Series: Hardware-based firewalls for various environments, from small offices to large data centers.
    • VM-Series: Virtual firewalls optimized for private and public cloud environments.
    • CN-Series: Container firewalls for protecting containerized applications and Kubernetes environments.
    • K2-Series: Firewalls designed for industrial and SCADA environments.
  • Panorama: Centralized management platform for consolidated policy creation and device management for NGFWs.
  • Threat Intelligence and Analysis:
    • WildFire: Cloud-based threat analysis service for zero-day threat mitigation.
    • Advanced Threat Prevention: Subscription service for blocking exploits, malware, and command-and-control traffic.
  • GlobalProtect: Network security for endpoints providing secure VPN access for remote users.

Prisma: Cloud Security

  • Prisma Access: Cloud-delivered security platform for extending consistent security policies to remote users and branch offices.
  • Prisma Cloud: Comprehensive cloud-native security platform for visibility, compliance, and protection across the cloud environment.
    • Compute Security: Protects workloads in containers, serverless functions, and VMs.
    • Network Security: Secures cloud networks with microsegmentation and threat prevention.
    • Identity and Access Security: Manages and secures user access to cloud resources.
    • Compliance and Governance: Ensures continuous compliance with industry standards.
  • Prisma SaaS: Security for SaaS applications providing visibility and control over SaaS usage, protecting data and preventing threats.

Cortex: Security Operations

  • Cortex XDR: Extended detection and response platform integrating data from network, endpoints, and cloud for comprehensive threat detection and response.
  • Cortex Data Lake: Scalable and secure cloud-based data storage for integrating and normalizing data from Palo Alto Networks security products.
  • Cortex XSOAR: Security orchestration, automation, and response platform for automating incident response workflows.
