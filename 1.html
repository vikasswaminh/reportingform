Newly Added Topics in CCNA and CCNP Courses

CCNA (Cisco Certified Network Associate)

The CCNA 200-301 exam has been updated to Version 1.1, effective from August 20, 2024. Here are some of the newly added topics:

  • Generative AI and Machine Learning:
    • Understanding the basics of AI and machine learning.
    • The role of AI in networking and how it can be utilized for network management and automation.
  • Cloud Network Management:
    • Introduction to cloud networking concepts.
    • Configuring and managing cloud-based networking solutions.
  • Enhanced Security Fundamentals:
    • New security protocols and measures.
    • Advanced threat detection and mitigation techniques.
  • Advanced Automation and Programmability:
    • More in-depth focus on automation tools and techniques.
    • Expanded coverage of network programmability with Python and APIs.

CCNP (Cisco Certified Network Professional)

The CCNP exams have also seen updates reflecting current industry trends and technologies:

  • Advanced Network Security:
    • In-depth exploration of modern security challenges.
    • Strategies for securing complex network environments.
  • Software-Defined Networking (SDN):
    • Principles and applications of SDN.
    • Implementing SDN solutions in enterprise networks.
  • Advanced Routing and Switching:
    • Enhancements in routing protocols like OSPF and BGP.
    • Advanced switching techniques and configurations.
  • Cloud Networking and Virtualization:
    • Detailed study of cloud networking architecture.
    • Integration of virtualized network functions (VNF) in the cloud environment.
  • Network Automation and Programmability:
    • Expanding on automation frameworks.
    • Utilizing scripting and APIs for network management and configuration.
